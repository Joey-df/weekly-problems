给定一个数组，任务是找到所需的最小运算次数，其中两个相邻元素从数组中移除并替换为它们的和，从而将数组转换为非递增数组
例子：
Input: A[] = {1, 5, 3, 9, 1}
Output: 2
Replacing {1, 5} by {6} modifies the array to {6, 3, 9, 1}
Replacing {6, 3} by {9} modifies the array to {9, 9, 1}
